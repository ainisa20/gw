# OPC公司注册助手 - 使用手册

## 简介

OPC公司注册助手是一款基于AI的智能表单填写辅助工具，能够自动识别当前操作页面，提供简洁的操作指导，帮助用户快速完成企业注册流程。

## 系统要求

- **操作系统**：Windows 10/11（64位）
- **网络**：需要联网（访问Dify AI服务）
- **运行环境**：无需安装Python或其他依赖，双击即可运行

## 安装与配置

### 1. 获取应用

将 `OPC公司注册助手.exe` 放到任意目录。

### 2. 首次启动

双击运行 `OPC公司注册助手.exe`，首次启动会自动在exe同目录生成 `.env` 配置文件，并弹出提示窗口。

### 3. 编辑配置

用记事本或其他文本编辑器打开 `.env` 文件，填入以下必填项：

```
# Dify API 地址（必填）
difyApi=http://your-dify-server:8889

# Dify API Token（必填）
difyToken=app-your-token-here

# 表单页面地址（可选，已有默认值）
# formUrl=https://example.com
```

保存后重新双击 exe 即可正常使用。

## 核心功能

### 1. 截图求助（最常用）

点击右侧的 **「截图求助」** 红色按钮，应用会：

1. 自动截取右侧页面当前状态
2. AI识别当前处于哪个操作步骤
3. 匹配知识库中的操作指南
4. 返回简洁的操作指导（编号列表，每步一句话）

**示例回复：**

> 📋 第3步 · 选择开办业务办理并点击办理入口
> 匹配依据: 页面显示"开办企业一窗通"
>
> 1. 在页面中找到 **"开办"** 按钮并点击
> 2. 选择 **"经营主体设立登记"**
> 3. 点击 **"办理"** 按钮进入下一步
>
> ⚠️ 需要先完成实名认证才能进入办理

### 2. 文字提问

在底部输入框中直接输入问题，例如：

- "这个页面怎么操作？"
- "法人信息怎么填？"
- "股东出资比例怎么分配？"

### 3. 图片上传

点击 📎 按钮上传截图，配合文字描述一起提问。

### 4. 语音输入

点击 🎤 按钮开始录音，再次点击停止，自动转为文字填入输入框。

## 常见问题

### Q: 启动后提示"请先配置"

A: 这是首次启动，请按提示编辑 `.env` 文件，填写 `difyApi` 和 `difyToken` 后重启应用。

### Q: 右侧页面加载空白

A: 检查网络连接，确认能正常访问政务网站。部分网站需要特定浏览器，本应用基于Edge内核。

### Q: 截图求助没反应

A: 截图功能需要右侧页面完全加载完成后再使用。如果页面还在加载中，请等待几秒后重试。

### Q: 修改了 .env 配置不生效

A: 修改 `.env` 后需要关闭并重新打开应用。

### Q: 应用运行缓慢

A: 应用基于Chromium内核渲染，正常内存占用约300-500MB，属于正常范围。建议关闭其他占用内存较大的程序。

## 注意事项

- `.env` 文件包含API密钥，请注意保密，不要发送给他人
- 应用需要联网使用，离线状态下AI功能不可用
- 右侧表单页面的操作与普通浏览器一致，可直接点击、输入、提交
- AI提供的操作指导仅供参考，请以实际页面要求为准
