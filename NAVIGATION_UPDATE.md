# 右侧浏览器导航优化

## 新增功能

### 导航控制栏
在URL输入框左侧添加了4个导航按钮：

```
[◀ 后退] [前进 ▶] [↻ 刷新] [⌂ 主页] [URL输入框...........] [加载]
```

## 功能说明

### 1. 后退按钮（◀ 后退）
- **功能**: 返回到上一个访问的页面
- **启用条件**: 当前页面有历史记录
- **快捷方式**: 相当于浏览器的"后退"
- **使用场景**: 点击链接后想返回上一页

### 2. 前进按钮（前进 ▶）
- **功能**: 前进到下一个页面（如果之前后退过）
- **启用条件**: 有前进历史记录
- **快捷方式**: 相当于浏览器的"前进"
- **使用场景**: 后退后又想回到刚才的页面

### 3. 刷新按钮（↻ 刷新）
- **功能**: 重新加载当前页面
- **启用条件**: 始终可用（只要有页面）
- **快捷方式**: 相当于F5或Ctrl+R
- **使用场景**:
  - 页面加载失败
  - 表单提交后重新填写
  - 内容更新后查看最新状态

### 4. 主页按钮（⌂ 主页）
- **功能**: 一键回到默认表单页面
- **默认页面**: `config.json` 中配置的 `formUrl`
- **启用条件**: 始终可用
- **使用场景**:
  - 迷路了想回到起点
  - 重新开始填写流程
  - 快速重置到初始页面

### 5. URL输入框 + 加载按钮
- **功能**: 手动输入URL访问其他网站
- **增强**:
  - 页面导航时自动更新URL
  - 支持回车键快速加载
  - 自动添加 `http://` 前缀（如果缺失）

## 智能特性

### 1. 自动启用/禁用导航按钮
```python
# 页面加载完成后自动更新
def _update_nav_buttons(self):
    self.btn_back.Enable(self.right.CanGoBack())
    self.btn_forward.Enable(self.right.CanGoForward())
```

**效果**:
- 没有历史记录时，"后退"按钮自动禁用（灰色）
- 没有前进历史时，"前进"按钮自动禁用（灰色）
- 页面每次加载后自动检查更新

### 2. URL自动同步
```python
# 页面导航时自动更新URL输入框
def on_right_navigating(self, evt):
    url = evt.GetURL()
    self.url_input.SetValue(url)
```

**效果**:
- 点击链接后，URL输入框自动显示当前页面地址
- 用户可以复制当前URL分享给他人

### 3. 错误处理
```python
def on_right_error(self, evt):
    print(f"[right] 加载错误: {evt.GetString()}")
```

**效果**:
- 网络错误时在终端显示日志
- 不会阻塞应用运行
- 用户可以点击"刷新"重试

## 使用场景示例

### 场景1: 点击错误链接
```
1. 用户在表单页面点击了某个链接
2. 页面跳转到其他网站
3. 用户想返回：点击"◀ 后退"按钮
4. 回到表单页面继续填写
```

### 场景2: 页面加载失败
```
1. 网络不稳定，页面加载不完整
2. 点击"↻ 刷新"按钮
3. 重新加载当前页面
```

### 场景3: 重新开始
```
1. 用户填写了很多步骤，想重新开始
2. 点击"⌂ 主页"按钮
3. 立即回到默认表单页面
```

### 场景4: 后退后又想前进
```
1. 点击"◀ 后退"回到上一页
2. 发现上一页不对，还是刚才的页面好
3. 点击"前进 ▶"回到刚才的页面
```

### 场景5: 手动访问其他网站
```
1. 用户复制了一个政府网站链接
2. 粘贴到URL输入框
3. 点击"加载"或按回车
4. 右侧加载新的网站
```

## 技术实现

### 使用的WebView API
```python
# 历史记录控制
self.right.CanGoBack()      # 检查是否可以后退
self.right.GoBack()         # 后退
self.right.CanGoForward()   # 检查是否可以前进
self.right.GoForward()      # 前进

# 页面控制
self.right.Reload()         # 刷新
self.right.LoadURL(url)     # 加载URL
self.right.GetCurrentURL()  # 获取当前URL
```

### 事件绑定
```python
# 页面加载完成
self.right.Bind(wx.html2.EVT_WEBVIEW_LOADED, self.on_right_loaded)

# 页面正在导航
self.right.Bind(wx.html2.EVT_WEBVIEW_NAVIGATING, self.on_right_navigating)

# 页面加载错误
self.right.Bind(wx.html2.EVT_WEBVIEW_ERROR, self.on_right_error)
```

## 已知限制

### 1. WebView限制
- 某些网站可能禁止在WebView中加载
- 部分JavaScript功能可能受限
- 证书错误页面无法自定义

### 2. 平台差异
- **macOS**: 使用WKWebView，功能完整
- **Windows**: 使用Edge WebView2，功能完整
- **Linux**: 使用wxPython默认WebView，功能可能受限

### 3. 历史记录
- 历史记录仅在应用运行期间保留
- 关闭应用后历史记录清空
- 不支持浏览器级别的"历史记录"
