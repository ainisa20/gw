# 右侧导航功能 - 快速使用指南

## 新增界面

```
┌─────────────────────────────────────────────────────────┐
│ [◀ 后退][前进 ▶][↻ 刷新][⌂ 主页] [https://....] [加载] │
└─────────────────────────────────────────────────────────┘
```

## 4秒速成

### 问题：点击错误链接后无法返回？
**解决**：点击 **◀ 后退** 按钮

### 问题：页面加载失败？
**解决**：点击 **↻ 刷新** 按钮

### 问题：想重新开始？
**解决**：点击 **⌂ 主页** 按钮

### 问题：后退后又想回去？
**解决**：点击 **前进 ▶** 按钮

## 按钮功能详解

| 按钮 | 快捷说明 | 何时使用 |
|------|---------|---------|
| **◀ 后退** | 返回上一页 | 点错链接了 |
| **前进 ▶** | 前进到下一页 | 后退后又想回去 |
| **↻ 刷新** | 重新加载当前页 | 页面加载失败 |
| **⌂ 主页** | 回到默认表单 | 迷路了想重来 |

## 智能特性

✅ **自动禁用/启用**
- 没有历史记录时，"后退"按钮自动变灰
- 没有前进历史时，"前进"按钮自动变灰

✅ **URL自动同步**
- 点击链接后，URL输入框自动显示当前地址
- 可以随时复制当前URL

✅ **工具提示**
- 鼠标悬停按钮会显示功能说明

## 使用场景

### 场景1：填表时点错了
```
1. 在表单页面不小心点了某个链接
2. 跳转到其他页面了
3. 点击 "◀ 后退" 回到表单
```

### 场景2：网络卡顿
```
1. 页面加载到一半卡住了
2. 点击 "↻ 刷新" 重新加载
3. 等待页面加载完成
```

### 场景3：想重新开始
```
1. 填了很多步骤，想重头来过
2. 点击 "⌂ 主页" 按钮
3. 立即回到初始表单页面
```

### 场景4：访问其他网站
```
1. 复制一个政府网站链接
2. 粘贴到URL输入框
3. 按 Enter 或点击 "加载"
```

## 常见问题

### Q: 为什么后退按钮是灰色的？
A: 当前页面没有历史记录，无法后退

### Q: URL输入框为什么会自动变化？
A: 页面导航时自动同步，方便你看到当前地址

### Q: 如何快速回到默认表单？
A: 点击 "⌂ 主页" 按钮，一键直达

## 技术细节

- **WebView API**: 使用wxPython的html2.WebView
- **历史记录**: CanGoBack()/CanGoForward()检测
- **事件监听**: EVT_WEBVIEW_LOADED/NAVIGATING
- **平台支持**: macOS(WKWebView), Windows(Edge WebView2)
